How to use Autodesk Navisworks

To run Autodesk Navisworks in another language

  1. Right-click the Autodesk Navisworks desktop shortcut, and click Properties on the context menu to open the Autodesk Navisworks Properties dialog box
  2. On the Shortcut tab, enter a space in the Target field after ..\roamer.exe, and then enter one of the following arguments:
    • -lang en-US for English localization
    • -lang zh-CN for Chinese (PRC) localization
  3. Click OK to save the changes.
